Bee JS

Bee-js is Bee's complementary JavaScript library. It is the technology underpinning the swarm-cli and bee-dashboard tools and is a powerful tool for building completely decentralised apps.

For more information on how to develop with Bee without blowing all your BZZ, read this guide

See the bee-js documentation for detailed information on using and installing the library.
